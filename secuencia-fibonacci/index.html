Secuencia fibonacci

Hacer una funcion cuyo resultado sea la secuencia fibonacci (0,1,1,2,3,5,8,13...)

Funcion js

(Inspecciona el navegador para ver la funcion ejecutandose) 
                
                     function fibonacci(iterations){
                        let aux     = 1;
                        let result  = [0];

                        for(let i = 0 ; i < iterations-1 ; i++){
                            aux = (result[i-1] || 0) + aux;
                            result.push(aux)
                        }

                        return result;
                    }
