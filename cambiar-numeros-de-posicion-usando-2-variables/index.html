Cambiar variables sin usar auxiliar

Teniendo 2 variables X y Y cambiarlas de posicion, que X tenga el valor de Y sin usar una tercera variable. Este ejercicio tiene 2 posibles soluciones, una usando sumas y restas y otra usando XOR sin embargo XOR no esta disponible en todos los lenguajes por lo que lo dejo solo de manera teorica

Funcion js

(Inspecciona el navegador para ver la funcion ejecutandose) 
                        
                            let X  = 10;
                            let Y  = 15;

                            X = X + Y;//25
                            Y = X - Y;//10
                            X = X - Y;//15

                            //XOR

                            X ^= Y;
                            Y ^= X;//10
                            X ^= Y;//15
