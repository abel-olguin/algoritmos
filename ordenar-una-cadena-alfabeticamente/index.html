Ordenar una cadena alfabeticamente

Dada una cadena de texto ordenarla de manera alfabetica en forma ascendente, lo importante a recordar es la funcion substr(donde_empieza,cantidad_de_caracteres)

                     function orderString(str){
                        e.preventDefault();

                        for(let i = 0 ; i < str.length ; i++){
                            if(str[i] > str[i+1]){
                                str = str.substr(0,i) + str[i+1] + str[i] + str.substr(i+2,str.length-i-2);

                               i = -1;
                            }
                        }
                        return str;
                    }
