Verificar si una cadena es un palindromo

Dada una cadena de texto verificar que es o no es un palindromo.

Funcion js

                
                     function checkPalindrome(str){
                        for(let i = 0 ; i < str.length/2 ; i++){
                            if(str[i] !== str[str.length -1 -i]){
                                return false;
                            }
                        }

                        return true;
                    }
