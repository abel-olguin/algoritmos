Ordenar un array de numeros de menor a mayor

Dado un array de numeros en desorden ordenarlos de menor a mayor ej. [5,4,3,2,1] = [1,2,3,4,5]

Funcion js

(Inspecciona el navegador para ver la funcion ejecutandose) 
                        
                             function order(arr){
                                for(let i = 0 ; i < arr.length ; i++){
                                    if(arr[i] > arr[i+1]){
                                        let aux     = arr[i];
                                        arr[i]      = arr[i+1];
                                        arr[i+1]    = aux;

                                        i = -1;
                                    }
                                }

                                return arr;
                            }
