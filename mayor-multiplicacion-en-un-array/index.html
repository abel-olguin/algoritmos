Obtener el mayor resultado de multiplicar 2 numeros en un array

Este ejercicio se puede dividir en 2 variantes:

  • Obtener el mayor resultado de multiplicar 2 numeros en un array que sean adyacentes ej. [1,2,3,4,5] 1*2 = 2, 2*3 = 6 ... 4*5 = 20, por lo tanto la respuesta sera 20
  • Obtener el mayor resultado de multiplicar 2 numeros en un array sin importar si son o no adyacentes ej. [5,1,2] 5*1 = 5, 5*2 = 10, 1*2 = 2, por lo tanto la respuesta es 10

Funcion js

(Inspecciona el navegador para ver la funcion ejecutandose) 
                        
                            /**
                             * cuando es adyacente
                             */
                             function productAdjacentResult(inputArray) {
                                let result = null;
                                for(let i = 0; i < inputArray.length; i++){
                                    let aux = inputArray[i]*inputArray[i+1];
                                    result =  aux > result || result === null?aux:result;
                                }

                                return result;
                            }
                            /**
                             * cuando no es adyacente
                             */
                            function productResult(inputArray) {
                                let result = null;

                                for(let i = 0; i < inputArray.length; i++){
                                    for(let j = i + 1 ; j < inputArray.length ; j++){
                                        let aux = inputArray[i]*inputArray[j];
                                        result =  aux > result || result === null?aux:result;
                                    }
                                }

                                return result;
                            }
